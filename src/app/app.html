Angular Skeleton Loader Library

This library helps you create skeleton placeholders in your Angular apps.

Skeletons improve user experience by showing visual placeholders while data is being fetched.

Apply skeletons to any element: titles, paragraphs, buttons, images, and lists.

Each card below demonstrates a combination of title, subtitle, text, additional info, and button skeletons.
